# React-native Monorepo Template

## CLI

```sh
npx react-native@latest init ProjectName
```

## Lib

- pnpm
- rollup

## Verify Physical Device

```sh
adb devices.
```

## Run App

```sh
adb -s <device name> reverse tcp:8081 tcp:8081

npm run android
```
